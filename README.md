# trio-take-home

App that fetches data from [Trio](http://trio.dev/) Mock McDonalds.

## Stack:
- React Native
- Expo
- Typescript

## External Libraries:
- [react-query](https://github.com/tanstack/query)
- [rn-placeholder](https://github.com/mfrachet/rn-placeholder)
- [axios](https://github.com/axios/axios)
